# strk20 privacy pool — live dashboard

Live onchain KPIs for the canonical STRK20 privacy pool on Starknet mainnet.
Aggregate metrics only, by design — no per-user attribution.

**Live:** https://welttowelt.github.io/strk20-pool-dashboard/

The page self-refreshes; the underlying data is regenerated from public
Starknet RPC. This repo holds only the rendered site (the tracker that
produces it is maintained separately).
